# 📝 Sticky Notes JS 

An interactive, user-friendly note-taking app designed to help users capture ideas quickly. This project focuses on efficient DOM manipulation and responsive UI design.

## 🚀 Live Demo
[Try the Sticky Notes App here!](https://suhas-46.github.io/Sticky-Notes-JS/)

## 🛠️ Built With
- **HTML5:** Semantic structure for a clean web experience.
- **CSS3:** Custom styling with Flexbox and box-shadows for a realistic note appearance.
- **Vanilla JavaScript:** Logic for dynamic note creation and interface interactivity.
- **GitHub Pages:** For global hosting and deployment.

## ✨ Features
- Instantly add new notes to your workspace.
- **Responsive Layout:** Notes wrap perfectly on mobile and desktop screens using CSS Flexbox.
- **Modern UI:** Minimalist design with soft shadows and clean typography.
